# 🍽️ Restaurant Management System (C++ OOP)

A console-based **Restaurant Management System** developed in **C++** using **Object-Oriented Programming (OOP)** concepts. This project simulates restaurant operations such as staff management, menu handling, customer order processing, and bill generation.

---

## 📖 Project Overview

The Restaurant Management System is designed to demonstrate the practical implementation of Object-Oriented Programming principles in C++. It provides an interactive console application where users can:

- View restaurant staff information
- Display the complete restaurant menu
- Place customer orders
- Generate a detailed bill with tax calculation
- Demonstrate OOP concepts through real-world examples

---

## ✨ Features

- 👨‍💼 Staff Management
  - Manager Information
  - Head Manager Information

- 🍴 Menu Management
  - Display complete menu
  - Item categories
  - Item prices

- 🛒 Customer Order System
  - Select menu items
  - Add multiple items
  - Calculate total amount
  - Calculate 10% tax
  - Generate final bill

- 💰 Billing System
  - Order summary
  - Itemized receipt
  - Tax calculation
  - Grand total
